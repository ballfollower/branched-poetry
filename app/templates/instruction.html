About the programme

The programme is used to create poems together. Each user can add his own verse to a given poem. The resulting poem branches into added verses, hence the name of the programme.

Usage

The programme supports four different operations:
  • Choosing an existing verse
    Click the "existent matches" list item to add a verse to the current poem.
  • Creating a new verse
    Enter the text of the new verse in the "new verse" edit box and press ENTER to add the verse to the current poem. The edit field can also be used to filter the "existent matches" list.
  • Starting over
    Press the "start over" button to restart the poem.
  • Downloading a poem
    Press the "download" button to download the current poem as a text file.
